# Code of Conduct

This project adopts the **[Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** verbatim.

The full text and translations are available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Concerns about behaviour in this project's spaces (issues, PRs, discussions) can be reported to **kennville@gmail.com**. All reports will be reviewed and investigated promptly and fairly. The maintainer is obligated to respect the privacy and security of the reporter.

## Scope

The Code applies in all project spaces and when an individual represents the project in public.
